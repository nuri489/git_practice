# Git Practice

---
### 배운것

- mkdir : 폴더 생성

- cd : 이하의 디렉토리로 이동

- touch : 파일 생성

- git init : 저장소 만들기

  > 해당 디렉토리에 저장소를 만든다.

- git add : 현재 상태 추적

  > 파일의 변경사항을 추적 함
  >
  > git add . 로 해당 디렉토리의 모든 파일의 상태를 추적함

- git commit  : 현재 상태 저장

  > 변경된 사항을 이력에 추가함
  >
  > git commit -m "ASD" 로 ASD라는 코멘트를 남길 수 있음

- git remote add origin 주소 : 원격 저장소의 주소를 저장함

  > 깃허브 저장소의 주소를 저장함

- git push origin master : 저장된 저장소로 변경된 사항을 전송함
